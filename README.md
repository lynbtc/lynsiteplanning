# siteplanning

site qualification tool for LYN modular sauna system.

## the system

one sauna. seven modules. twentyonehundred sites.

| module | description | dimensions |
|--------|-------------|------------|
| m1 sauna | the core. panoramic glass. | 3.3 × 2.2 × 2.5 m |
| m2 swimming bridge | direct water access. | 1.5 × 2.0 × 0.3 m |
| m3 changing room | transitional space. skylit. | 1.5 × 2.0 × 2.5 m |
| m4 thermal module | bitcoin mining heat source. | site-dependent |
| m5 outdoor shower | wall-mounted. cooling. | 0.5 × 0.1 × 2.0 m |
| m6 storage | robes. towels. slippers. | 0.6 × 0.3 × 1.5 m |
| m7 cold plunge | self-contained basin. | 1.5 × 2.0 × 2.5 m |

## flow

- private / professional
- outdoor / indoor site types
- module configuration
- thermal module (m4) integration
- lead capture via Netlify Forms

## stack

- single HTML file
- vanilla JS
- no dependencies
- Netlify Forms

## configuration

edit `CONFIG` and `MODULES` in index.html.
